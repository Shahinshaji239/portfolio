Ott Streaming Platfrom

Ott Streaming Platfrom

Cineverse is a full-stack OTT streaming web application that replicates the essential features of platforms like Netflix. It allows users to browse a collection of movies, view details, maintain a watchlist, and track viewing history. The frontend is built using React, while Django powers the backend and API layer.

The user experience starts with a combined login and signup interface, allowing seamless transitions between the two. User authentication is handled securely through Django, and once logged in, users are directed to the home page where movies are dynamically loaded from the backend using RESTful APIs.

Users can explore available movies, view detailed information including thumbnails and descriptions, and play the selected videos. Cineverse also features personalized functionality such as the ability to add or remove movies from a watchlist and track previously watched content automatically. Additionally, users have access to account management features like changing passwords and securely logging out.

The project is designed to be responsive, ensuring a consistent experience across various devices. The backend is built with Django, providing a robust framework for handling user authentication, data management, and API interactions. The frontend leverages React's component-based architecture for a dynamic and interactive user interface.

Personal Bookmark Site

Personal Bookmark Site

The Personal Bookmarking Site is a lightweight React application that allows users to save and manage their favorite website links efficiently. The project features a combined login and signup interface on a single page, with buttons to toggle between the two forms. Once logged in, users can add bookmarks by entering a title and a URL. To maintain simplicity and avoid clutter, the application restricts each user to a maximum of five bookmarks. If the limit is exceeded, an error message is displayed to inform the user.

Each bookmark is displayed with its title, clickable URL, and the timestamp of when it was added. Users can view their bookmarks in a clean, paginated layout, which ensures smooth navigation even when multiple entries are present. A search bar is provided to filter bookmarks based on the title or URL, making it easy to locate specific entries quickly. Users can also edit or delete existing bookmarks whenever needed.

All data is stored using the browser’s local storage, eliminating the need for a backend or database. This approach makes the app fast, responsive, and ideal for offline usage. The logout option clears the session and returns the user to the login screen, ensuring personal data is not accessible afterward.

Project cloning
